# BlogSpace

A full-stack MERN Blog Platform where users can create, edit, delete, and comment on blog posts.

## Features

- User Registration & Login
- JWT Authentication
- Create Blog Posts
- Edit Own Posts
- Delete Own Posts
- Add Comments
- Delete Own Comments
- Responsive Design
- RESTful API

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT
- bcryptjs

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/saravanabhavank23/Blogspace.git
```

### 2. Install backend dependencies

```bash
cd backend
npm install
```

### 3. Create a `.env` file inside the `backend` folder with the required environment variables.

### 4. Start the backend

```bash
npm run dev
```

### 5. Open the frontend

Open `frontend/index.html` using Live Server.
